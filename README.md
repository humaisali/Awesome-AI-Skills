## About The Project

As AI coding agents like **Claude**, **GitHub Copilot**, and **Antigravity IDE** become deeply integrated into our workflows, the need for specialized, context-rich instructions has skyrocketed. I collected these skills from thousands of repositories and websites because finding high-quality, reliable AI instructions took too much effort.

My goal is to democratize access to elite AI engineering tools. By organizing over 2,000 distinct skills into a single, clean repository, developers everywhere can instantly plug specialized knowledge into their local AI agents.

### What are AI Agent Skills?

An "Agent Skill" is not just a text prompt. It is a modular package (usually containing a `SKILL.md` file) that includes:

- **Metadata:** Name, description, and authorship.
- **System Instructions:** Deep, specialized rules on how the AI should behave (e.g., "always use React server components").
- **Assets:** Reference scripts, documentation, or templates that the AI can read to execute tasks perfectly.

### Why Use Skills Over Prompts?

While prompts are short, one-off commands, **Skills** act as permanent brain upgrades for your agent. When you install a skill, your agent automatically knows *how* to approach complex, multi-step domain problems without needing to be re-prompted every time.

## Getting Started

Injecting these skills into your workflow is incredibly simple.

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/humaisali/awesome-ai-skills.git
   ```
2. **Find Your Skill:**
   Navigate into the relevant domain folder (e.g., `Development & Engineering Skills/`).
3. **Attach to your Agent:**
   Open the target folder and provide the `SKILL.md` file to your AI.

**For a comprehensive, step-by-step tutorial on configuring these skills automatically in Claude and Antigravity IDE, please read the [USER_GUIDE.md](USER_GUIDE.md).**

## FAQ

**Q: Will these skills work with ChatGPT?**
A: Yes! While they are optimized for Agentic frameworks (like Claude, Cursor, or Antigravity) that can read markdown natively, you can simply copy and paste the contents of the `SKILL.md` file into ChatGPT as a Custom Instruction or a prompt.

**Q: I found a skill that doesn't work. What should I do?**
A: AI models change rapidly. If a skill's instructions are outdated, please open an Issue using the Bug Report template, or better yet, submit a Pull Request with an updated prompt!
