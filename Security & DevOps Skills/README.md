## Overview

The **Security & DevOps Skills** category is designed to assist engineers in building, deploying, and securing applications with absolute confidence. These skills instruct your AI agent on the precise methods for configuring cloud environments, automating Continuous Integration and Continuous Deployment (CI/CD) pipelines, and performing rigorous security audits. By leveraging these modules, you can confidently automate complex infrastructure tasks while ensuring strict compliance with modern security and operational standards.

---

## Subcategories

This category is divided into two critical operational domains:

### Cyber Security
Transform your agent into a diligent Security Analyst and Penetration Tester. These skills cover automated vulnerability scanning, penetration testing basics, secure coding practices (aligning with OWASP Top 10), comprehensive threat modeling frameworks (like STRIDE), and incident response strategies. Utilizing these skills ensures your agent actively looks for and patches vulnerabilities like SQL injection, Cross-Site Scripting (XSS), and insecure direct object references before they reach production.

### DevOps & Infrastructure
Master cloud deployments, infrastructure as code (IaC), and operational automation. Equip your AI with the skills to write safe Terraform configurations, orchestrate Kubernetes clusters, set up robust GitHub Actions pipelines, optimize Docker container images for size and security, and manage resources across AWS, Azure, and Google Cloud Platform. These instructions enforce zero-downtime deployment strategies and strict secret management.

---

## How to Use

1. Navigate to the subcategory that matches your infrastructure or security requirements.
2. Browse the folders to locate the specific skill module you require.
3. Attach or copy the contents of the `SKILL.md` file into your agent's context (for example, into Claude, Antigravity, or Cursor).
4. Initiate your prompt and watch your agent execute tasks with domain-expert precision, constrained by the best practices defined in the skill.
