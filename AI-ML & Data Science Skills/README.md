## Overview

The **AI-ML & Data Science Skills** category is a meticulously curated collection of agentic skills designed to elevate your AI's capabilities in data manipulation, algorithmic problem-solving, and machine learning infrastructure. Whether you are building intelligent agents, orchestrating complex Large Language Model workflows, or performing deep statistical data analytics, these skills provide your AI with the precise context, constraints, and methodologies required to excel.

By utilizing these skills, you transform your general-purpose AI into a specialized Data Scientist, Machine Learning Engineer, or AI Architect. The instructions within these modules ensure that your agent follows industry standards for data governance, model evaluation, and computational efficiency.

---

## Subcategories

This category is divided into the following specialized domains:

### Agents & LLMs
This subcategory contains advanced instructions for designing, building, orchestrating, and evaluating Autonomous Agents and Large Language Models. Equip your AI to understand multi-agent architectures, state management, long-term memory structures, prompt optimization techniques, and orchestration frameworks such as CrewAI, AutoGen, and LangChain. These skills are essential for building reliable, agent-driven applications that avoid hallucinations and maintain logical coherence over long reasoning chains.

### Data Science & Analytics
Focuses on the entire data pipeline, including data wrangling, statistical analysis, A/B testing frameworks, and advanced data visualization techniques. These skills teach your agent the best practices for handling large, unstructured datasets, performing rigorous root-cause analysis, and generating actionable business insights. It includes instructions for working with Python data science libraries like Pandas, NumPy, and Scikit-learn, ensuring your agent writes clean, performant analytical code.

### Machine Learning
Dedicated to the core principles of ML engineering and operations. From model training and evaluation pipelines to MLOps, deep learning architectures, and inference optimization techniques. Use these skills to guide your agent in building robust, production-ready machine learning systems, handling hyperparameter tuning, model deployment strategies, and managing tensor operations safely.

---

## How to Use

1. Navigate to the subcategory that matches your current task or project phase.
2. Browse the folders to locate the specific skill module you require.
3. Attach or copy the contents of the `SKILL.md` file into your agent's context (for example, into Claude, Antigravity, or Cursor).
4. Initiate your prompt and watch your agent execute tasks with domain-expert precision, constrained by the best practices defined in the skill.
