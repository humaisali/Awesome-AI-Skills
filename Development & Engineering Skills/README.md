## Overview

The **Development & Engineering Skills** category is the core library for software developers and systems architects. It contains rigorous, enterprise-grade instructions that enforce clean code, scalable architecture, and modern development paradigms. By applying these skills, you ensure that your AI agent writes code that is not merely functional, but also highly maintainable, secure, and optimized for performance. This section is strictly curated to avoid anti-patterns and enforce language-specific idioms.

---

## Subcategories

This category covers the entire software development lifecycle across various platforms:

### Backend & APIs
Equip your AI with the architectural knowledge to build resilient, scalable backend systems. This section covers REST and GraphQL API design, relational and NoSQL database schema optimization, microservices architecture, secure authentication flows (OAuth, JWT), and server-side logic across languages like Node.js, Python, Go, and Java. These skills ensure your backend infrastructure can handle production-level traffic and edge cases.

### General Coding & Architecture
Focuses on fundamental, language-agnostic software design principles. Give your agent a deep understanding of Gang of Four Design Patterns, SOLID principles, algorithm optimization, memory management, and complex system architecture. These skills act as a foundational layer to ensure that whatever language your agent writes in, the underlying logic is structurally sound and extensible.

### Mobile Development
Instructions specialized for native iOS (Swift), Android (Kotlin), and cross-platform frameworks such as React Native and Flutter. These skills ensure your agent strictly follows platform-specific UI guidelines, hardware integration best practices, performance tuning, and mobile memory management to prevent memory leaks and UI thread blocking.

### Web Development
The ultimate toolkit for frontend mastery. Teach your AI the latest best practices in modern web frameworks (React, Next.js, Vue, Svelte), vanilla JavaScript, HTML and CSS semantics, web accessibility (a11y) standards, and complex state management. These skills guarantee that your agent builds responsive, performant, and accessible web applications that meet modern browser standards.

---

## How to Use

1. Navigate to the subcategory that matches your current development stack or architectural need.
2. Browse the folders to locate the specific skill module you require.
3. Attach or copy the contents of the `SKILL.md` file into your agent's context (for example, into Claude, Antigravity, or Cursor).
4. Initiate your prompt and watch your agent execute tasks with domain-expert precision, constrained by the best practices defined in the skill.
